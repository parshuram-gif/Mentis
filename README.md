The target : I wanted something to read which doesn't feel too long and is informative. 
There are too many apps but I wanted to read according to my preferences.
So I made this. 
This is a simple site where you get to read 1 information about anything based on a preset category allocated by percentage.
How it works? : I had given it some predefined category and assigned some percentage to each category.
It uses the API and give a summary of 1 info from the category. 
You also get an option to read the full detailed version also.
I had also added a streak feature. 
It also has archive which is saved locally. 
You can select from the category before you generate it or you can keep it upto random selection.
It generate only one article a day you come back again it's still there. It refreshes the next day.
